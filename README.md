# bike-sharing-demande-in-paris

Prédiction du nombre en vélos nécéssaires dans une ville.

Nous comptons utiliser le dataset Bike Sharing Demand sur Kaggle, qui contient les données d’utilisation de vélos en libre-service (nombre de trajets par heure/jour) dans un endroit précis ainsi que des variables explicatives comme la météo, la saison, le jour de la semaine et l’heure.


Le lien du dataset : https://www.kaggle.com/competitions/bike-sharing-demand/data
